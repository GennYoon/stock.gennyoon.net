# 배당 투자를 위한 나만의 스마트 비서 앱 (Smart Dividend Assistant App)

## 개요
배당 투자를 위한 스마트 비서 앱은 사용자가 배당주 투자 현황을 쉽게 파악하고, 목표를 설정하며, 장기적인 투자 여정을 효율적으로 관리할 수 있도록 돕는 웹 애플리케이션입니다.

## 목표
- 사용자가 배당주 투자 현황을 한눈에 파악할 수 있도록 지원
- 사용자가 합리적인 배당금 목표를 설정하고 달성 계획을 세울 수 있도록 지원
- 장기적인 배당 투자 여정에서 사용자에게 동기를 부여하고 현명한 의사결정을 돕는 정보 제공

## 주요 기능

### 1. 나만의 배당 포트폴리오 관리 (통합 대시보드)
- 보유 주식 등록: 미국/한국 배당주 등록, 종목명, 티커, 매수 단가, 수량, 매수일 입력
- 실시간 가치 반영: 현재 시장 환율 반영, 원화 가치 실시간 표시
- 성과 요약 대시보드: 예상/실제 수령 배당금, 총 배당 수익률, 투자 원금, 평가액, 평가 손익 표시
- 시각화된 배당 내역: 월별, 분기별, 연도별 배당금 수령 내역 차트

### 2. 초고배당주 분석 및 시뮬레이션
- 초고배당주 리스트: 시장 상장된 초고배당주 목록 제공
- 총 수익률 시뮬레이션: 가상 매수 시 총 수익률 계산, 주가 하락 시나리오 포함

### 3. 배당금 목표 달성 계산기
- 목표 배당금 달성 시뮬레이션: 연간 목표 배당금 달성을 위한 필요 투자액 계산
- 목표 수량에 따른 예상 배당금: 특정 주식의 목표 보유 수량별 예상 연간 배당금 계산

### 4. 원금 회수율 추적 기능
- 원금 대비 배당금 누적 현황: 투자 원금 대비 총 배당금 및 회수율(%) 표시
- 원금 회수까지 남은 기간 예측: 원금 회수 예상 소요 기간 계산

### 5. 배당 투자 정보 허브
- 최신 뉴스 및 리포트: 관심 배당주 및 전체 배당 시장 관련 뉴스, 공시, 리포트 제공
- 시장 트렌드 정보: 배당 투자에 유용한 주요 경제 지표 및 시장 트렌드 정보 제공

## 기술 스택
- 프론트엔드: Next.js 15, React 19, TypeScript, Tailwind CSS v4
- UI 컴포넌트: shadcn/ui, Radix UI
- 상태 관리: React State Management
- 데이터 소스: 외부 API (Polygon.io 등) 및 웹 스크래핑
- 데이터베이스: PostgreSQL 또는 SQLite (로컬 개발용)

## 제약 사항
- 초기 개발은 미국/한국 배당주에 한정
- 외부 데이터 연동은 웹 스크래핑 또는 무료/공개 데이터 활용
- 사용자 보유 주식 정보의 자동 연동은 초기 개발 범위에서 제외
- 증권사별 API 연동은 채택하지 않음
